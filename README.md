# Hotel Booking Data Analysis

## Description
This project analyzes hotel booking data to explore customer behavior, booking patterns, and the main factors that affect hotel reservations.

## Dataset
The dataset includes information related to hotel bookings, such as:
- Lead time
- Number of guests (adults, children, babies)
- Length of stay
- Booking changes
- Cancellations
- Average daily rate (ADR)

## Objectives
- Analyze booking patterns
- Calculate summary statistics
- Understand customer behavior
- Identify important trends in reservations

## Features
- Data cleaning and handling missing values
- Statistical analysis
- Data interpretation
- Extracting insights from the dataset
- Data visualization

## Technologies Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Key Insights
- Customers often book several weeks or months in advance.
- Most stays are concentrated in specific periods and patterns.
- Booking changes and cancellations vary across customers.

## How to Run
1. Download the dataset file.
2. Open the notebook in Jupyter Notebook.
3. Run all cells step by step to reproduce the analysis.
